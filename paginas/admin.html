Gestión de Categorías

Administra las categorías y subcategorías del catálogo

¿Cómo funciona?
Para crear una Categoría Principal:
  1. Escribe el nombre (Ej: "Salas", "Comedores", "Roperos")
  2. Deja el segundo campo VACÍO
  3. Clic en "Guardar"
Para crear una Subcategoría:
  1. Escribe el nombre (Ej: "Sofacamas", "Reclinables")
  2. En el segundo campo, escribe a qué pertenece (Ej: "Salas")
  3. Clic en "Guardar"
El nombre es obligatorio
Déjalo vacío para crear una categoría principal. Escribe un nombre para crear una subcategoría.
